User Manual
MUN Moderator 2016
About
MUN Moderator is a set of Web Applications built specially for Kumarans Model United Nations by the KMUN Tech Team. It is built completely on web technologies to enable moderators to easily monitor and assist in committee proceedings by using any device with a modern browser. MUN Moderator 2016 can also provide detailed statistics and scoring reports for each delegate. It takes MUN software to the next-level.
Features
MUN Moderator 2016 features many features to make managing a MUN session a breeze.
  • Completely web-based and platform independent - Even works on tablets and phones
  • Simple and easy to use with a good design
  • Integrated with live crisis feed
  • Works offline and syncs when there is internet connectivity
  • Statistics for easy scoring of delegates and analysis of sessions
  • Caucus, GSL and Voting management
Voting and Roll Call
Voting and roll call is simple to use. Select the required radio button in the "Voting" tab and click "Start Vote!". Roll Call needs to be done before Simple Majority and Resolution Vote can be done.
